# mm2d

Robot planar pushing code.

## Install

```bash
# Clone the repository
git clone https://github.com/adamheins/mmpush

# Install basic dependencies
cd mmpush
poetry install

# Activate the virtualenv
poetry shell 
```

## Organization

* `mmpush`: Shared utilities.
* `scripts`: Scripts for pushing stuff.
